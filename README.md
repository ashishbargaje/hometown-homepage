# 🏡 Hometown Homepage – Pune

A responsive tourism-inspired landing page showcasing my hometown, **Pune**, built using **HTML5** and **CSS3** as part of a solo project. The project was based on a Figma design challenge and customized with my own content, visuals, and branding.

## 🌐 Live Demo

🔗 https://avbargajetech-hometown-homepage.netlify.app/

## ✨ Features

- Clean and modern landing page design
- Hero section featuring Pune's iconic places
- Personalized content highlighting the heart of Pune ❤️
- Three featured local attractions
- Custom "Meet Your Guide" section with hover feature
- Semantic HTML5 structure
- Pure CSS styling (no frameworks)

## 🛠️ Built With

- HTML5
- CSS3
- Flexbox
- Google Fonts

## 📚 What I Learned

While building this project, I practiced:

- Semantic HTML
- CSS Flexbox
- Typography
- Spacing and visual hierarchy
- Color palette selection
- Working from a Figma design
- Deploying a static website using Netlify

## 🎯 Project Goal

The objective of this project was to recreate a provided Figma design while adding my own creativity through:

- Personalized content
- Custom copywriting
- Local attractions from Pune
- Personal branding
- Improved UI polish
